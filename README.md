### Sobre
WEBAPI feita basicamente com JavaScript e Node. É possivel ler, adicionar, alterar e excluir os dados dos usuários.


### Tecnologias e Integrações
- JavaScript
- Node

### Uso e instalação
- Apenas executar o arquivo no terminal, utilizando o comando "node".
